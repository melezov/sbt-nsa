Possible private options include:
  -P:<plugin>:<opt>                 Pass an option to a plugin
  -Ybrowse:<phase>                  Browse the abstract syntax tree after <phase>
  -Ycheck:<phase>                   Check the tree at start of <phase>
  -Yclosure-elim                    Perform closure elimination
  -Ycodebase <codebase>             Specify the URL containing the Scala libraries
  -Ydebug                           Output debugging messages
  -Ydead-code                       Perform dead code elimination
  -Ydetach                          Perform detaching of remote closures
  -Ydoc                             Generate documentation
  -Yinline                          Perform inlining when possible
  -Ylinearizer:<Ylinearizer>        Linearizer to use (normal,dfs,rpo,dump)
  -Ylog:<phase>                     Log operations in <phase>
  -Ylog-all                         Log all operations
  -Yno-imports                      Compile without any implicit imports
  -Yno-predefs                      Compile without any implicit predefined values
  -Yshow-trees                      Show detailed trees when used in connection with -print:phase
  -Yskip:<phase>                    Skip <phase>
  -Ysqueeze:<Ysqueeze>              if on, creates compact code in matching (on,on,off)
  -Ystatistics                      Print compiler statistics
  -Ystop:<phase>                    Stop after phase <phase>
  -Ywarn-dead-code                  Emit warnings for dead code
  -Ycasetags:<Ycasetags>            test integer tags for case classes (on,off)
  -Yself-in-annots                  Include a "self" identifier inside of annotations
